alumnos: julio rodriguez, leandro ceballos, mijael ayala
Proyecto desarrollado para la materia: Aplicaciones para Dispositivos Móviles. La aplicacion consiste en un catalogo interactivo de videojuegos, que permite gestionar un carrito de compras y una lista de favoritos.
se desarrollo utilizando Vue CLI.
Diseño responsive adaptable a dispositivos móviles implementado con CSS.
se usaron componentes como: gamecard.vue y gamedetail.vue comunicados de forma limpia mediante props y eventos personalizados $emit.
Enrutamiento entre pantallas (Catálogo Principal y Vista de Carrito/Favoritos) a través de Vue Router.
Persistencia y sincronización de datos con la memoria de la instancia de Vue y LocalStorage(Permite agregar, editar cantidad de unidades y eliminar elementos).
Control de consistencia en el formulario de Checkout (validación regex de email y longitud de nombre) antes del procesamiento.
Cómo ejecutar el proyecto localmente:
npm run dev, Ctrl + Clic al enlace.
para terminar CTRL + C
